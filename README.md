🛒 Daily Basket - Full Stack MERN Grocery Shopping Platform

A modern and responsive grocery shopping application built with the MERN Stack. The platform allows users to browse products, manage their cart, securely authenticate using JWT, and complete purchases through Stripe payment integration.

🚀 Features

- 🔐 JWT Authentication & Authorization
- 🛒 Shopping Cart Management
- 💳 Secure Stripe Payment Gateway
- 📱 Fully Responsive Design
- 🥬 Product Browsing
- 🔍 Search & Filter Products
- 📦 Order Management
- 👤 User Profile Management
- ⚡ Fast React-Based Frontend
- 🌐 RESTful API Architecture

🛠 Tech Stack

Frontend

- React.js
- React Router
- Axios
- CSS / Tailwind CSS

Backend

- Node.js
- Express.js
- JWT Authentication
- Stripe API

Database

- MongoDB
- Mongoose
